# Graph-Algoritms
Консольная программа, которая реализует иерархию классов в соответствии с вариантом задания: транспортное средство, автомобиль, поезд, экспресс. Пользователь может добавлять объекты в хранилище, просматривать список объектов, искать объект по имени, редактировать информацию о нём и удалять объекты из массива.
## Реализовано
• Поддержка ориентированных и неориентированных графов.
 • Чтение графа из трех различных форматов:
 • список рёбер;
 • список смежности;
 • матрица смежности.
 • Вычисление степеней вершин (или полустепеней входа/выхода для орграфов).
 • Поиск компонент связности для неориентированных графов.
 • Поиск компонент слабой связности для ориентированных графов.
 • Поиск компонент сильной связности для ориентированных графов.
 • Вычисление характеристик графа:
   • радиуса;
   • диаметра;
   • множества центральных вершин;
   • множества периферийных вершин.
 • Реализация алгоритма Флойда—Уоршелла для поиска кратчайших путей между всеми парами вершин.
 • Реализация алгоритма Дейкстры для поиска кратчайшего пути между двумя заданными вершинами.
 • Реализация алгоритма Прима или Крускала для построения минимального остовного дерева (если ты выполняла этот пункт).
 • Поддержка работы через командную строку (CLI) с параметрами запуска.
 • Возможность вывода результатов как в консоль, так и в файл.
 • Реализована встроенная справка по использованию программы (-h)
